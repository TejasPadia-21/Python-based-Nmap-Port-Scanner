# Python-based-Nmap-Port-Scanner

Building Cyber Tools with Python + Nmap 🐍📡
🔍 Just completed a Python-based Nmap Port Scanner that can scan TCP, UDP, or both — directly from the terminal!
💡 Features:
✅ Validates IP addresses
 ✅ Performs fast or comprehensive scans (TCP, UDP, or full)
 ✅ Displays open ports, services, and state in a clean format
 ✅ Checks if nmap is installed before running
This project gave me hands-on experience with:
⚙️ nmap automation using Python’s nmap module
🔁 Real-world user interaction with validation
📡 Network mapping and reconnaissance techniques
💻 Defensive mindset by understanding how attackers scan systems
